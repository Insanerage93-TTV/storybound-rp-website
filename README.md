# StoryBound RP Website

This is the official website for **StoryBound Roleplay (SBRP)**.  
Built with **React + Tailwind CSS**.

## 🚀 Development
```bash
npm install
npm run dev
```

## 📦 Build for production
```bash
npm run build
```

## 🌍 Deployment
Can be hosted on Vercel, Netlify, or Cloudflare Pages.
